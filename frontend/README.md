# Nurfia Fashion - Frontend

This is the React-based frontend for the Nurfia Fashion eCommerce platform.

## 🚀 Key Components
- **Product Gallery**: Interactive image preview with thumbnail support.
- **Cart & Wishlist**: Persistent state management for shopping convenience.
- **Admin Layout**: Integrated dashboard for site management.
- **Responsive Navigation**: Adaptive header and footer components.

## 🛠 Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```

## 📦 Build
To create a production build:
```bash
npm run build
```

The output will be available in the `dist` directory.
